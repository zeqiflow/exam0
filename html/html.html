1. HTML

1.1
  • 超文本标记语言（Hyper Text Markup Language）
  • 是一种标记语言，不是编程语言。
  • 使用各种标签来描述网页。
  • 使用.html 或者 .htm作为文件后缀。
  • 只有<body> </body>中间的内容会在网页中显示。
  • <head>之间的内容会在浏览器的顶部显示。

    • HTML不受到缩进的严格控制，只以标签的起止作为边界

2. CSS

2.1 简介

CSS 的使用提高了代码的复用性，有点像编程语言中的类，一次定义重复使用。

  • CSS指层叠样式表（CascadingStyle Sheets）。
  • 样式定义了如何显示HTML元素。
  • 样式表用来存储样式。
  • CSS 由选择器和声明构成。
  • 多个标签可以同时定义样式。
  • 一个标签样式可以被定义多次。
  • 语法格式参见css文件。

3. JavaScript

3.1 简介

JavaScript是网页编程语言，控制了网页的行为。

  • HTML中的脚本必须位于<script></script>之间。可以放在head和body中，但是通常放在head中。
  • 也可以把需要重复使用的脚本放在外部， 并在<script>的src属性中声明地址。外部的脚本不能包含script标签。
